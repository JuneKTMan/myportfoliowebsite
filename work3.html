virtual Assistant by Python

Built a virtual Assistant by Python

In this project, I have learned how to use different python module and internal library to build a python Virtual Assistant. And this is the practice for the AI customer chart bot

Technologies

* Python 3.9.8
* Pycharm Profession 2021.3.1
* SQLite 3.7.15
* Python module pip install: 1. pip install pyttsx3
2. pip install pywhatkit
3. pip install wikipedia
4. pip install pyjokes
If you are using Python 3.7 or above, there will have error when using the pyttsx3 module, you can choose to downgrade the pyttsx3
pip install pyttsx3==2.71
or
engine= pyttsx3.init('dummy')

Instruction to start

1. Turn on your microhpone of your device Run the virtualAssistant_jun/main.py.(you can download the file from my git repositories)
2. Virtual Assistant can do: Play your favorite song!
Just say "Jun, play Taylor Swift's song."
3. Virtual Assistant can do: Search who!
Just say "Jun, who is x?"
4. Virtual Assistant can do: Tell the time!
Just say "Jun, tell me the time."
5. 4. Virtual Assistant can do: Tell you some jokes!
Just say "Jun, tell me some jokes."
