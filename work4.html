Projectile Simulator by Java

2D Animation built by java

This Java program will demonstrate classes, inheritance, and threads. And the animation is simulating the motion of projectile movement with physical logic.

Technologies

* java 17.0.1
* eclipse

Instruction

1. Clone the repo https://github.com/JuneKTMan/projectile_simulator.git
2. Open the project folder by eclipse or other IDE.
3. Run the Project folder > projectile_test> mainpanel_projectile.java
